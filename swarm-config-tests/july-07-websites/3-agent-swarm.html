Deploy Your 3-Agent Swarm

Get started with your own cognitive symphony in minutes.

1. Install ruv-swarm 

npm install -g ruv-swarm
ruv-swarm init 3-agent-symphony

2. Configure Agents 

const swarm = new RuvSwarm({
  agents: [
    { type: 'researcher', model: 'lstm-attention' },
    { type: 'designer', model: 'tcn-creative' },
    { type: 'developer', model: 'n-beats-code' }
  ],
  topology: 'hierarchical',
  parallelExecution: true
});

3. Launch Swarm 

ruv-swarm start --agents 3 --parallel
